# ペイントアプリ 要件定義書

## 1. 概要

| 項目 | 内容 |
|------|------|
| プラットフォーム | Webブラウザ |
| ターゲットユーザー | 初心者・一般ユーザー |
| UIコンセプト | シンプルで直感的な操作 |

## 2. 機能要件

### 2.1 描画ツール

| ツール | 説明 |
|--------|------|
| ペン（ブラシ） | フリーハンド描画 |
| 消しゴム | 描画した部分を消去 |
| 塗りつぶし | 閉じた領域を色で塗りつぶす |
| 図形ツール | 線、四角形、円の描画 |

### 2.2 ブラシカスタマイズ

- **サイズ変更**: スライダーで太さを調整
- **ブラシ形状**: 丸、四角などの形状を選択可能

### 2.3 カラー機能

- **カラーパレット**: 予め用意された色から選択

### 2.4 ファイル操作

| 機能 | 説明 |
|------|------|
| 保存 | 作業中のデータをローカルに保存（プロジェクトファイル） |
| 読み込み | 保存したプロジェクトファイルを開く |
| 画像エクスポート | PNG形式でダウンロード |

### 2.5 履歴機能

| 機能 | 説明 | ショートカット |
|------|------|----------------|
| Undo | 直前の操作を取り消し | Ctrl+Z |
| Redo | 取り消した操作をやり直し | Ctrl+Y |

## 3. 画面構成

```
┌─────────────────────────────────────────┐
│  ヘッダー（タイトル、ファイル操作）       │
├──────┬──────────────────────────────────┤
│      │                                  │
│ツール│         キャンバス                │
│バー  │    （ウィンドウサイズ連動）        │
│      │                                  │
│ 色   │                                  │
│パレット                                  │
│      │                                  │
└──────┴──────────────────────────────────┘
```

### 3.1 ヘッダー
- アプリケーションタイトル
- ファイル操作ボタン（新規、保存、読み込み、エクスポート）
- Undo/Redoボタン

### 3.2 サイドバー（左）
- 描画ツール選択
  - ペン
  - 消しゴム
  - 塗りつぶし
  - 図形（線、四角、円）
- ブラシ設定
  - サイズスライダー
  - 形状選択
- カラーパレット

### 3.3 キャンバスエリア
- ウィンドウサイズに連動してリサイズ
- 描画領域

## 4. 技術スタック

| 技術 | 用途 | 選定理由 |
|------|------|----------|
| HTML5 Canvas | 描画処理 | ブラウザ標準の描画API |
| Vanilla JavaScript | ロジック実装 | 依存関係なし、シンプル、学習コスト低 |
| CSS3 | UIスタイリング | モダンなUI実現 |
| LocalStorage | 設定保存 | ブラウザ標準API |
| File API | ファイル操作 | ファイル保存・読み込み |

## 5. 非機能要件

### 5.1 対応ブラウザ
- Google Chrome（最新版）
- Mozilla Firefox（最新版）
- Microsoft Edge（最新版）
- Safari（最新版）

### 5.2 レスポンシブ対応
- ウィンドウリサイズに追従
- 最小幅: 800px

### 5.3 パフォーマンス
- 描画操作は60FPSを目標
- Undo履歴は最大50回分を保持

- レイヤー機能

- カラーピッカー
- テキスト入力
- 画像インポート
- クラウド保存
